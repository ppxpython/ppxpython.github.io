Path 的语法
从 django2.1 以后用 path 之前用的是 url
标准语法
Path (route,view,name=None,**kwargs)
Route:
端口以后 url 的地址，到 / 结束，表示路径
View:
表示路径匹配成功后，需要调用的视图，必须是个函数如果是 class 的话，必须要用 as_view () 函数装换为函数
name：
表示别名
**kwargs：
表示一个字典

前面两个是必选项
